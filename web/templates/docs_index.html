Getting Started

The M3U8 Downloader Bot is a Telegram bot that allows you to download and process M3U8 video streams. It's designed to be easy to use and highly configurable.

ℹ️

New to M3U8? M3U8 is a file format used for multimedia playlists, primarily for streaming audio and video. It's commonly used by streaming services to deliver content.

Quick Start

  1. Start a chat with the bot on Telegram
  2. Send an M3U8 URL to the bot
  3. Wait for the bot to download and process the video
  4. Receive the processed video file
